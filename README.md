# pump-fun-sniper-bot

## setting
### config env
```commandline
    pip install -r requirements.txt
```
### config rpc and wallets
```commandline
    configure config.json
```

### usage
```commandline
    step 1: choose your wallet
    step 2: choose your rpc endpoint
    step 3: config amount(sol) you want to buy each time
```

### run 
```commandline
    python main.py
```
